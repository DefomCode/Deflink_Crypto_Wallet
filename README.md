# DefLink Crypto Wallet (DCW)

Автономный CLI-менеджер холодных крипто-кошельков с шифрованием AES-256.
Поддерживает сети Ethereum (ETH) и Tron (TRX).

> ⚠️ Альфа-версия. Используйте на свой страх и риск. Всегда делайте бэкап vault.json.

## Возможности

- 🔐 Хранение приватных ключей с шифрованием AES-256-GCM
- 🌐 Мультичейн: ETH + TRX в одном менеджере
- 💰 Отслеживание баланса в нативной валюте и USD
- 📤 Отправка транзакций с интерактивным подтверждением
- 📜 История транзакций с пагинацией
- ⌨️ Гибридный UX: работает с 0, 1, 2 или 3 аргументами + полный интерактивный режим
- 🗑️ Массовое удаление кошельков с подтверждением каждого
- 🔤 Автодополнение имён кошельков в Zsh (Tab)
- 🏠 Полностью автономный — никаких внешних серверов, никакой телеметрии

## Установка

Через pipx (рекомендуется):
```bash
    pipx install git+https://github.com/defomcode/Deflink_Crypto_Wallet.git
```
Поддержка Tron требует библиотеку tronpy, которая на некоторых системах ставится отдельно:
```bash
    pipx inject dcw "coincurve==20.0.0" tronpy
```
## Использование

Создание кошелька (интерактивный выбор сети):
```bash
    dcw create
```
Создание TRX-кошелька напрямую:
```bash
    dcw create my_trx --net TRX
```
Проверка баланса:
```bash
    dcw balance my_wallet
```
Отправка средств (гибридный UX):
```bash
    dcw pay my_wallet 0xПолучатель... 0.001
```
Список всех кошельков с кэшированными балансами:
```bash
    dcw list
```
История транзакций:
```bash
    dcw history
```
Удаление нескольких кошельков:
```bash
    dcw delete wallet1 wallet2
```
Все команды поддерживают как однострочный режим, так и полностью интерактивный.

## Поддерживаемые сети

| Сеть       | Нативная монета | Стандарт токенов | Формат адреса |
|------------|-----------------|------------------|---------------|
| Ethereum   | ETH             | ERC-20           | 0x...         |
| Tron       | TRX             | TRC-20           | T...          |

Один адрес в каждой сети принимает и нативные монеты, и токены.

## Хранение данных

- Хранилище: ``~/.config/dcw/vault.json`` — зашифрованные приватные ключи
- Кэш/История: ``~/.config/dcw/dcw.db`` — база данных SQLite
